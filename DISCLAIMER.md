# GoAxonAI Legal Disclaimer

## AI-Assisted Productivity Tool Notice

> ⚖️ **GoAxonAI acts as an AI-assisted productivity tool. Applications are submitted only with explicit user action.**

## Data Sources

GoAxonAI aggregates job listings from **publicly available APIs** provided by third-party job boards. We do **NOT**:

- Scrape company career portals
- Use private or undocumented APIs
- Access data behind authentication
- Store copyrighted content (job descriptions are truncated)

## API Sources Used

All job data is obtained from **official, documented APIs**:

| Source | API Documentation | Terms |
|--------|-------------------|-------|
| **Adzuna** | [developer.adzuna.com](https://developer.adzuna.com/) | Official job aggregator API |
| **Arbeitnow** | [arbeitnow.com/api/job-board-api](https://www.arbeitnow.com/api/job-board-api) | Free public API |
| **Remotive** | [remotive.com/api-documentation](https://remotive.com/api-documentation) | Free public API |
| **The Muse** | [themuse.com/developers](https://www.themuse.com/developers/api/v2) | Free public API |
